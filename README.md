# 📝 BackStasks

BackStasks is a simple and intuitive task management web application that allows users to organize their tasks using categories and keep track of their progress.

---

## 🚀 Features

* 📂 Create and delete categories with custom colors
* ✅ Add, view, and delete tasks
* 🏷 Assign tasks to categories
* ✔ Mark tasks as completed
* 🎨 Clean and modern UI with improved UX design

---

## 🛠 Technologies Used

* ASP.NET Core MVC
* Razor Views
* Bootstrap 5
* C#
* HTML / CSS

---

## 📸 Overview

BackStasks provides a simple interface where users can:

* Manage categories on the left panel
* Add new tasks easily
* View tasks in a clean card-based layout

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/backstasks.git
```

2. Navigate to the project folder:

```bash
cd backstasks
```

3. Run the project:

```bash
dotnet run
```

4. Open your browser at:

```
https://localhost:5001
```

## 📌 Usage

* Add a category with a custom color
* Create tasks and assign them to categories
* Track your tasks and keep things organized
